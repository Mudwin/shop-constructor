# Описание проекта

**Концепция**: конструктор интернет-магазинов для независимых авторов и создателей.

**Целевая аудитория**: художники, писатели, музыканты, дизайнеры, блогеры, digital-художники.

**Особенности**: продавец может создать собственный интернет-магазин, предназначенный только для своих товаров. Это могут быть как цифровые товары (музыка, рисунки, дизайны, шрифты и т.д.), так и физические товары (сувениры, картины и т.д.)
